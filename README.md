## jsNote

This application uses an Express.js back end to save and retrieve note data from a JSON file.

![licensebadge](https://img.shields.io/badge/license-CC0_1.0_Universal-blue)

#### Requirements

- node.js
- npm

#### Installation

    npm i

#### Usage

    npm start
    navigate: http://localhost:3001/
